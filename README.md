# ids568-milestones

Setting up from scratch

You are going to clone the repo by:
1) git clone https://github.com/jkaur2026/ids568-milestones.git
cd ids568-milestones

You are going to do write out git clone and you are going to get the github repo link paste that click enter and then you are going to write cd and then the name of your repo 

Then you are going to initialize the python environment and if you end up going with venv which is the virtual environment you are going to write python3 -m venv venv which is the name of your virtual environment but doenst have to be named venv. Then you click enter and then you are going to write out source venv/bin/activate.

Once that is done then you are going to move on to install the dependencies. You are going to write pip install --upgrade pip then pip install numpy pandas scikit-learn pytest then enter and then pip freeze > requirements.txt. You could also do pip install -r requirements.txt
